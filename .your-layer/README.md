.your-layer/
SpecKit-SSD-SDLC custom layer. Everything here is owned by this framework
and will never conflict with upstream spec-kit files.

Directories

commands/    — sk.* slash commands (wrappers + originals)
templates/   — document templates for architecture, ADR, PHR, contracts
scripts/     — shell scripts for ADR/PHR creation, upstream reconciliation
governance/  — checkpoint rules, quality gates, SDLC flow, roles
